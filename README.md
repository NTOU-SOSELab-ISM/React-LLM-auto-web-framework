## 整體更新 & chatGPT4o 的實驗報告：

- 實驗步驟：
  - 在本地先架好 React & Vite 環境
  - 請 LLM (GPT4o) 生出最初始的程式碼，生到好為止
  - 接下來進行每次迭代：
    - 人工檢查網頁是否符合最初始的使用者需求，如果是，停止迭代
    - 如果不是，請詳細描述不符合的使用者需求，丟給GPT4o，讓他整體做更新，進行下一次迭代。
